## 二、待完成功能（建议优先级）

### 🔴 高优先级（毕设必须有）

| 功能 | 说明 |
|------|------|
| **商品评价展示** | 商品详情页下方展示其他用户的评价列表，目前只能写评价，看不到 |
| **物流信息展示** | 订单详情页内显示物流轨迹，后端数据已生成但前端没有渲染 |
| **订单取消** | 待发货状态下，用户可以主动取消订单并退回余额 |
| **库存管理** | 商品增加库存字段，下单时扣减，售罄后显示"已抢光"并禁止购买 |
| **管理端会员管理** | 后台可查看所有用户列表、余额、积分、VIP 等级 |

### 🟡 中优先级（加分项）

| 功能 | 说明 |
|------|------|
| **商品搜索优化** | 当前搜索只过滤标题，增加按分类、价格区间筛选 |
| **管理端图表完善** | 数据看板增加订单趋势折线图、近 7 天销售额 |
| **拼团大厅实时状态** | 显示当前进行中的拼团、已凑几人/还差几人 |
| **商品图片上传** | 管理端新增/编辑商品时支持直接上传图片（目前需要填 URL）|
| **管理端发货填写单号** | 发货时填写快递单号，物流信息真实展示 |

### 🟢 低优先级（有时间再做）

| 功能 | 说明 |
|------|------|
| 收货地址默认标记 | 设置默认地址，下单自动选中 |
| 商品分享 | 生成商品海报/链接（可用截图模拟） |
| 消息通知 | 订单状态变更时前端弹出提示 |
| 密码修改 | 个人中心增加修改密码入口 |
| 注册时赠送新人券 | 注册成功自动发放"新人礼券"到账户 |
